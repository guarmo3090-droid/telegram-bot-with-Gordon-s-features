# AI Telegram Бот-Перекладач

Цей проєкт — це Telegram-бот, інтегрований з великою мовною моделлю (LLM) через OpenAI API. Бот виконує наперед задану роль професійного перекладача між українською та англійською мовами.

Проєкт реалізовано на платформі **.NET 8** з використанням архітектури `Microsoft.Extensions.Hosting` для забезпечення стабільної фонової роботи сервісу.

## Технології та бібліотеки
- **C# / .NET 8**
- **[Telegram.Bot](https://github.com/TelegramBots/Telegram.Bot)** — клієнт для роботи з Telegram Bot API.
- **[OpenAI](https://github.com/openai/openai-dotnet)** — офіційний клієнт для взаємодії з LLM.
- **Microsoft.Extensions.Hosting** — для організації фонової служби (BackgroundService).
- **Microsoft.Extensions.Configuration** — для безпечного зчитування конфігурації.

## Налаштування та запуск

### 1. Конфігурація 
Згідно з правилами безпеки, токени не зберігаються у вихідному коді. Для запуску бота необхідно створити файл `appsettings.json` у корені проєкту з таким вмістом:

```json
{
  "BotConfig": {
    "TelegramToken": "ВАШ_TELEGRAM_BOT_TOKEN",
    "OpenAiKey": "ВАШ_OPENAI_API_KEY",
    "SystemPrompt": "Ти - перекладач. Твоя задача - перекладати текст з української на англійську і з англійської на українську.\n\nПравила роботи:\n- якщо користувач пише українською, переклади англійською;\n- якщо користувач пише англійською, переклади українською;\n- не додавай пояснень, коментарів чи зайвого тексту;\n- зберігай зміст, тон і стиль оригіналу."
  }
}


# Відновлення залежностей
dotnet restore

# Запуск бота
